dashboards.ly faq and documentation

dashboards.ly is an open source web application for arranging plotly graphs into web dashboards. Check out an example or get started.

How do I create these graphs?
Dashboards.ly syncs with your online plotly account. Use the plotly web application or the plotly API to create graphs.
How do I update the dashboards?
Dashboards are composed of embedded plotly graphs. You can edit and update any of your plotly graphs in your online plotly account.You can also update charts programatically with one of plotly's API libraries by overwriting a graph with the same filename.
Can a graph be embedded in multiple dashboards?
Yes.
Is it possible to create dashboards programatically?
Yes - see the API section below.
How do I keep dashboards and plotly graphs private?
Plotly graphs can be public (graphs appear in search engines, the plotly feed, and your public profile), private (you, and viewers you grant access, must login to plotly to be able to view the graphs), or secret (graphs are accessible at a random URL and won't appear in search engines, the plotly feed, or your profile. Login is not required).
If you embed private graphs in a dashboard, you'll need to login to plotly to view them and you'll need to grant viewing access to other plotly users. This is what a plotly dashboard with private graphs looks like as a logged out user.
View this dashboard.

If you embed secret graphs in a dashboard, you or your viewers don't need to log in to plotly and the graphs and the dashboard are only accessible by direct link.

API

JSON Structure
Dashboards are described declaratively in JSON. For example, here is the JSON representation of this dashboard. 
{
    "rows": [
        [{"plot_url": "https://plot.ly/~benji.b/38"}, {"plot_url": "https://plot.ly/~benji.b/54"}],
        [{"plot_url": "https://plot.ly/~benji.b/62"}]
    ],
    "banner": {
        "visible": true,
        "backgroundcolor": "#3d4a57",
        "textcolor": "white",
        "title": "Quarterly Outlook",
        "links": [
            {"href": "https://google.com", "text": "Financials"},
            {"href": "https://google.com", "text": "Growth"},
            {"href": "https://google.com", "text": "Performance"}
        ]
    },
    "requireauth": false,
    "auth": {
        "username": "Acme Corp",
        "passphrase": ""
    }
}
rows is an array of arrays. Each inner array contains the URLs of the graphs that should be embedded in that row. In this example, the top two embedded graphs are the plots are https://plot.ly/~benji.b/38 and https://plot.ly/~benji.b/54 and are created by the plotly user "benji.b". To embed secret graphs, use the full secret URL with the share_key URL parameter, e.g. https://plot.ly/~chriddyp/1425?share_key=9Rh7y8eL9HO3LX1zJ6HS9i.
To enable passphrase authentication, set "requireauth" to true and supply a username and passphrase in "auth".
POST at https://dashboards.ly/publish
Arguments: dashboard - URL encoded dashboard JSON described above.
Returns: {"url": A_LINK_TO_YOUR_DASHBOARD}
Example of a raw request:

Python Example
View this dashboard. 
import plotly.plotly as py

## First, make some graphs to embed in the dashboard.
# See more examples: https://plot.ly/python
graph_url1 = py.plot({
    'data': [{'x': [1, 2, 3], 'y': [3, 1, 5]}],
    'layout': {
        'title': 'earnings',
        # graphs embedded in dashboards look best if the margins are tightened up
        'margin': {'l': 30, 'r': 30, 'b': 30, 't': 60}
    }
}, filename='dashboard/earnings',
   sharing='secret') # more about privacy settings: https://plot.ly/python/privacy

graph_url2 = py.plot({
    'data': [{'x': [1, 2, 3], 'y': [3, 40, 5]}],
    'layout': {
        'title': 'growth',
        'margin': {'l': 30, 'r': 30, 'b': 30, 't': 60}
    }
}, filename='dashboard/growth', sharing='secret')

graph_url3 = py.plot({
    'data': [{'x': [1, 2, 3], 'y': [10, 6l, 2]}],
    'layout': {
        'title': 'performance',
        'margin': {'l': 30, 'r': 30, 'b': 30, 't': 60}
    }
}, filename='dashboard/performance', sharing='secret')

## Now, create a dashboard
import requests

dashboard_json = {
    "rows": [
        [{"plot_url": graph_url1}, {"plot_url": graph_url2}],
        [{"plot_url": graph_url3}]
    ],
    "banner": {
        "visible": True,
        "backgroundcolor": "#3d4a57",
        "textcolor": "white",
        "title": "Quarterly Outlook",
        "links": []
    },
    "requireauth": False,
    "auth": {
        "username": "Acme Corp",
        "passphrase": ""
    }
}

response = requests.post('http://dashboards.ly/publish',
                         data={'dashboard': json.dumps(dashboard_json)},
                         headers={'content-type': 'application/x-www-form-urlencoded'})

response.raise_for_status()

dashboard_url = response.json()['url']
print('dashboard url: http://dashboards.ly{}'.format(dashboard_url))
# View here: http://dashboards.ly/ua-2jc39MpJCAouZ8fGgC3hBk
